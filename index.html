Featured Projects

Career Institute – A Laravel-Based CMS

Career Institute is a powerful Content Management System (CMS) built from scratch in Laravel. It is currently used by more than 12 institutes for managing their operations efficiently.

The system includes a fully dynamic front-end website, controlled entirely from the admin panel. It offers key modules such as leads management, admissions, expenses, classes, timetables, and fee management.

It also features certificate generation and verification, along with more than 10 types of reports to help institutes track and analyze their data effectively.

PHP Laravel MySQL jQuery Bootstrap

eShops – A Multi-Seller eCommerce Platform

eShops is a Laravel-based eCommerce platform built from scratch. It allows each seller to have their own fully dynamic shop on a separate subdomain, managed through an admin panel.

Key features include category and product management, the ability to resell other sellers’ products, seller subscriptions, and one-to-one account and transaction management for users. The system also provides a complete order management module.

The same functionality is available in the Flutter app, for which I developed optimized REST APIs with improved response times.

PHP Laravel MySQL jQuery Bootstrap

City Police Office FSD – Bail Management System

City Police Office FSD is a Laravel-based CMS built from scratch. It manages data for individuals released on bail and tracks their monthly follow-ups.

The system supports multiple user roles, each handling different types of data efficiently. It ensures organized record-keeping and smooth management of follow-up processes.

Laravel Ajax MySQL JavaScript Bootstrap

LinkGesture – SEO Link Management CMS

LinkGesture is a Laravel-based CMS built from scratch to manage SEO links efficiently. It allows publishers to add their blog website links along with pricing, while users can place their articles on these websites.

The system integrates Google Sheets API for seamless import and export of data. A Google crawler is also implemented to fetch website logos and other relevant information.

The admin panel includes a complete eCommerce workflow, featuring cart, checkout, and order management modules.

PHP Laravel Ajax MySQL jQuery Bootstrap

Mansuba-e-Amal (Citizen Portal - KP)

Mansuba-e-Amal is a Laravel-based complaint management system built from scratch. It integrates external APIs for division, district, VC, and NC data.

Citizens can submit public complaints, accessible to all users. Complaints go through multiple approval stages based on user roles. Each user is registered under a specific VC/NC and can only file complaints related to their area.
